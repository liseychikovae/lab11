# Совместная разработка через ngrok

Лабораторная работа Лисейчиковой Екатерины

1. Установить ngrok в Linux-окружении.
2. Запустить локальный сервис разработки.
3. Открыть туннель командой `ngrok http 8080`.
4. Передать HTTPS-адрес участнику совместной сессии.

В репозитории оставлены инструкции, чтобы не хранить персональные auth token и временные адреса туннелей.
