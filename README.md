# lab11

Лабораторная работа Лисейчиковой Екатерины

## Отчет по лабораторной работе XI

### Тема

Изучение совместной разработки с использованием ngrok.

### Цель работы

Цель работы - описать безопасный сценарий временного доступа к локальному сервису разработки.

### Используемое окружение

- основной целевой сценарий: Linux;
- язык: C++11;
- система сборки: CMake;
- дополнительные инструменты добавляются по теме конкретной лабораторной;
- персональные токены и временные адреса не сохраняются в репозитории.

### Структура проекта

- `formatter_lib` - базовая статическая библиотека форматирования;
- `formatter_ex_lib` - расширенная библиотека, использующая `formatter`;
- `solver_lib` - библиотека для решения квадратного уравнения;
- `hello_world_application` - демонстрационное приложение;
- `solver_application` - приложение для вывода корней уравнения;
- `tests` - модульные тесты, начиная с lab05;
- `docs` и `scripts` - дополнительные материалы поздних лабораторных.

### Ход выполнения

- добавлен документ `docs/ngrok.md`; 
- описана установка ngrok и запуск HTTP-туннеля;
- добавлен скрипт `scripts/dev_server.sh` для локального сервера на порту 8080;
- в отчете отмечено, что auth token и временные URL не хранятся в git;
- сохранены все файлы предыдущих лабораторных.

### Команды для проверки

```bash
bash scripts/dev_server.sh
ngrok http 8080
curl http://localhost:8080
```

### Домашнее задание

Домашнее задание выполнено внутри этого же репозитория. Подробное описание вынесено в `HOMEWORK.md`.

Кратко по выполнению:

- ngrok используется как временный туннель для демонстрации проекта другому участнику;
- локальный сервер запускается отдельным скриптом, чтобы не привязываться к IDE;
- секреты ngrok должны задаваться локально, а не попадать в репозиторий;
- документация объясняет порядок действий для совместной сессии.

### Результат

Репозиторий содержит понятный и безопасный сценарий совместной разработки через временный туннель.
