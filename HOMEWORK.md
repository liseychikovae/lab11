# Домашнее задание lab11

Лабораторная работа Лисейчиковой Екатерины

## Что требовалось сделать

Домашнее задание относится к теме лабораторной работы XI: изучение совместной разработки с использованием ngrok.

## Выполненные пункты

- ngrok используется как временный туннель для демонстрации проекта другому участнику;
- локальный сервер запускается отдельным скриптом, чтобы не привязываться к IDE;
- секреты ngrok должны задаваться локально, а не попадать в репозиторий;
- документация объясняет порядок действий для совместной сессии.

## Как проверить домашнее задание

```bash
bash scripts/dev_server.sh
ngrok http 8080
curl http://localhost:8080
```

## Пояснение по реализации

Проект оформлен так, чтобы его можно было перенести в Linux-окружение и собрать стандартными командами CMake. Файлы не завязаны на абсолютные пути macOS, персональные секреты не добавлены в git, а дополнительные инструменты подключаются через конфигурационные файлы конкретной лабораторной.
